```bash
$ cgit <operation>

Usage:
    cgit h: help (this output)
    cgit v: corgit version

    cgit s: status
    cgit e: push an existing repo
    cgit ph: push and commit
    cgit pl: fetch and pull

Setup:
    bash setup

Uninstall:
    bash un
```
